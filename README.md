## Design Philosophy

The application is designed with neurodivergent users in mind:
- **Calming color palette** - Soft blues, greens, and muted tones
- **Minimal distractions** - Clean, focused interface
- **Clear visual hierarchy** - Easy to understand and navigate
- **Smooth transitions** - Gentle animations that don't overwhelm
- **Accessible** - Keyboard navigation and focus indicators

## Getting Started

1. Simply open `index.html` in a modern web browser
2. No installation or build process required
3. All features work offline after initial load
